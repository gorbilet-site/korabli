# Горбилет · Прогулки по воде

Лендинг с картой причалов Петербурга и тестом подбора прогулки.

## Что в репозитории

```
index.html              ← ГОТОВЫЙ САЙТ — всё внутри, один файл
data/piers.json         ← причалы и акции (справочно)
data/quiz-actions.json  ← акции теста (справочно)
js/map.js, js/quiz.js   ← модули (справочно)
assets/pin-flame.svg    ← иконка причала
```

Для работы сайта достаточно одного `index.html` — карта, тест и все данные уже внутри. Папки `data/` и `js/` — это исходники для будущих правок.

## Шаг 1. Залить на GitHub
1. github.com → **+** (вверху справа) → **New repository**
2. Имя — любое (например `korabli`), **Public**, **Create repository**
3. **Add file → Upload files** → перетащите `index.html` (или весь распакованный архив)
4. **Commit changes**

## Шаг 2. Включить GitHub Pages
**Settings → Pages → Source: Deploy from a branch → Branch: main → / (root) → Save**

Через минуту вверху появится адрес: `https://ВАШ-ЛОГИН.github.io/имя-репозитория/`

## Карта

Карта работает на Leaflet и OpenStreetMap и не требует API-ключа. Метки привязаны к географическим координатам причалов; при переключении фильтра карта автоматически охватывает все видимые точки.

## Обновление акций
Все данные акций вшиты в `index.html`:
- блок `const PIERS=[...]` — причалы и их акции (для карты)
- блок `const QUIZ_ACTIONS=[...]` — акции теста с тегами

Меняйте нужную строку через карандаш ✏️ на GitHub.

## Геолокация «Вы здесь»
Работает только по HTTPS — GitHub Pages даёт его автоматически.
